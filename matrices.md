# 🗺️ Módulo 3: Vectores y Matrices (Generación de Terreno)

En programación, los **vectores** (o arreglos de una dimensión) nos sirven para guardar una lista de datos del mismo tipo en una sola variable. Pero si necesitamos algo más complejo, como una cuadrícula, usamos las **matrices** (arreglos de dos dimensiones, con filas y columnas).

En un juego de supervivencia, las matrices son perfectas para representar el mundo. Imagina un tablero de ajedrez: cada "casilla" (coordenada) puede contener información sobre el tipo de terreno que hay en ese lugar (agua, tierra, bosque, lava).

## 🎮 Ejercicio Práctico: El Mapa del Mundo
Para este ejemplo, hemos creado una matriz bidimensional (2D) de caracteres (`char[][]`). Cada letra representa un bioma distinto. Luego, usando ciclos anidados (un ciclo `for` dentro de otro `for`), el programa "dibuja" nuestro mundo en la consola, fila por fila.
